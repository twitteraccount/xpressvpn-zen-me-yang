ExpressVPN是在“ RAM磁盘”模式下运行所有​​服务器的少数VPN之一，这意味着您的所有数据在每次重新引导时都会被完全删除（RAM无法存储任何数据，因此一旦服务器切换后，一遍又一遍，您会话中的所有数据都会被完全擦除）。该公司将该设置称为TrustedServer功能。
